# 设计文档

## 信息结构
- 首页：欢迎页 + 进入用户管理入口。
- 用户页：创建区、详情区、列表区。

## 交互约定
- 创建用户成功后即时更新列表。
- 点击“查看”加载单个用户详情。
- 列表支持行内编辑并保存。
- 支持删除用户并即时移除列表项。
- 错误信息在表单区域就地显示。

## 视觉约定
- 基于 shadcn/ui 组件体系（Card/Table/Input/Button/Badge）。
- 明亮底色 + 渐变背景，突出卡片层级。

## 组件约定
- 输入框：单行名称输入。
- 按钮：支持 `default`、`secondary`、`outline`、`destructive`。
- 列表项：显示 `id`、`name`、`createdAt`，操作区包含查看/编辑/删除。
